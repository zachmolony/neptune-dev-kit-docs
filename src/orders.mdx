# Orders

The ```neptuneOrders``` hook provides a set of functions to create orders.

## Usage

```js
import { neptuneOrders } from 'neptune-kit';

const { createOrder } = neptuneOrders();
```

## API

### createOrder

The ```createOrder``` function creates an order. It accepts an order object as a parameter and returns a promise that resolves to the created order.

```js
const order = {
  cartId: '1',
  shippingAddress: {
    firstName: 'John',
    lastName: 'Doe',
    address1: '123 Main St',
    address2: 'Apt 1',
    city: 'New York',
    state: 'NY',
    zip: '10001',
    country: 'US',
  },
  billingAddress: {
    firstName: 'John',
    lastName: 'Doe',
    address1: '123 Main St',
    address2: 'Apt 1',
    city: 'New York',
    state: 'NY',
    zip: '10001',
    country: 'US',
  },
  paymentMethod: {
    id: '1',
    name: 'Visa',
    type: 'credit_card',
    last4: '4242',
    expMonth: 1,
    expYear: 2022,
  },
};

createOrder(order).then((order) => {
  console.log(order);
});
```
