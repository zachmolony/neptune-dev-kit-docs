# About Neptune

Neptune is a headless commerce platform that enables developers to build and scale commerce experiences. Neptune is built on top of the [Stripe](https://stripe.com) API and provides a set of tools to build and manage your commerce experience.

# Introduction

NeptuneKit is a collection of custom hooks to handle commerce logic and interaction with Neptune's platform. It's built on top of [React](https://reactjs.org/) and provides a set of hooks to handle common commerce use cases such as retriving products, interaction with a cart, and creating orders.
