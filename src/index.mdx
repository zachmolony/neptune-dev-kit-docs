# Getting Started

To get started with NeptuneKit in your Node.js project, simply add the dependency with NPM.

```bash
npm install neptune-kit
```

Once installed, you can import any of the hooks into your project:
```js
import { neptuneCart, neptuneProducts, neptuneOrders } from 'neptune-kit';
```
