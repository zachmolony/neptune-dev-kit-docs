# Getting Started

To get started with Neptune Dev Kit in your React project, simply add the dependency with NPM or Yarn.

```bash
npm install neptune-dev-kit
# or
yarn add neptune-dev-kit
```

Once installed, you can import any of the hooks into your project:

```js
import { useCart, useProducts, useOrders } from "neptune-dev-kit";
```
